# Симулятор поддержки Wildberries

Реалистичная веб-игра-симулятор работы оператора службы поддержки маркетплейса, максимально приближенная к реальным условиям регламентной поддержки.

## Описание

Игра предназначена для обучения, аттестации и стресс-тестирования операторов. Она воспроизводит системное давление, ограничения и когнитивную нагрузку, характерные для настоящей поддержки.

## Особенности

- Реалистичное поведение клиентов с различными типами поведения (агрессивные, запутанные, требовательные и т.д.)
- Строгая система шаблонов ответов, соответствующая регламенту поддержки
- CRM-система с информацией о клиентах, заказах и истории обращений
- Комплексная система KPI с оценкой корректности, регламента, времени реакции и удовлетворенности
- Интеграция с ИИ для генерации поведения клиентов и анализа ответов оператора

## Архитектура

### Компоненты:
- `App.jsx` - Главный компонент приложения
- `ChatInterface.jsx` - Интерфейс чата с клиентами
- `CRMSystem.jsx` - CRM-система с информацией о клиентах
- `KPIMonitor.jsx` - Мониторинг показателей эффективности

### Сервисы:
- `TemplateManager.js` - Управление шаблонами ответов
- `ClientGenerator.js` - Генерация клиентов и их поведения
- `KPISystem.js` - Система оценки показателей эффективности

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение в режиме разработки:
```bash
npm run dev
```

3. Приложение будет доступно по адресу http://localhost:3000

## Игровая механика

### Поток обращений
- Клиенты пишут в свободной форме с эмоциональными, неполными или противоречивыми сообщениями
- Поведение клиентов включает повторение требований, агрессию, ссылки на "закон", требования невозможного

### Действия оператора
- Выбор категории обращения
- Запрос уточняющих данных
- Выбор шаблона ответа
- Отправка сообщения
- Передача диалога супервайзеру

### Система оценки (KPI)
Каждое действие влияет на показатели:
- **Корректность** - соответствие шаблону и контексту
- **Регламент** - отсутствие запрещенных формулировок
- **Время реакции** - задержки снижают рейтинг
- **Эскалации** - лишние передачи штрафуются
- **Удовлетворенность клиента** - рассчитывается вероятностно

## Шаблоны ответов

Все ответы оператора строго выводятся из предоставленных шаблонов. При создании игры были использованы реальные шаблоны ответов поддержки Wildberries, включая:

- Уточнения
- Рассрочку
- Клиент не оформлял заказ
- Неизвестное списание
- Смена статуса
- Внутренние запросы
- Автоотмена
- Возврат по браку
- И другие категории

## Цель симуляции

Игра обучает и проверяет:
- Корректную классификацию обращений
- Выявление недостающих данных
- Выбор правильного шаблона
- Соблюдение регламентов и зон ответственности
- Устойчивость к давлению, агрессии и неопределенности
- Последовательность решений во времени

Игра системно наказывает за интуитивные, но нерегламентные действия, попытки "помочь по-человески" и выход за полномочия оператора.
